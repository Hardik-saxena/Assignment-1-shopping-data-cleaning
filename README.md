# Shopping Data Cleaning and Exploration using Pandas

## Objective
Learn Python basics and perform data exploration and cleaning using Pandas.

## Tasks Performed
- Loaded CSV dataset into Pandas DataFrame
- Explored dataset using:
  - head()
  - tail()
  - shape
  - columns
  - dtypes
- Handled missing values
- Removed duplicate rows
- Performed filtering and column selection
- Created derived column:
  - discount_amount
- Saved cleaned dataset as CSV

## Technologies Used
- Python
- Pandas
- NumPy
- Kaggle Notebook

## Files
- `shopping_data_analysis.ipynb`
- `cleaned_shopping_data.csv`

## Dataset
Shopping Dataset from Kaggle
